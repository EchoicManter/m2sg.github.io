404 Not Found!

404是一个非常常见的错误，他几乎出现在所有的网站中，本网站 采用的是HappyPaper式系统（不需要后端，只需要工程师修改前端的内容让网页改变的形式），会出现此错误是无法避免的。 在这里我们将会讲述404错误的详细情况。

404页面是客户端在浏览网页时，服务器无法正常提供信息，或是服务器无法回应，且不知道原因所返回的页面。据说在第三次科技革命之前，互联网的形态就是一个大型的中央数据库，这个数据库就设置在404房间里面。那时候所有的请求都是由人工手动完成的，如果在数据库中没有找到请求者所需要的文件，或者由于请求者写错了文件编号，用户就会得到一个返回信息：room 404 : file not found。404错误信息通常是在目标页面被更改或移除，或客户端输入页面地址错误后显示的页面，人们也就习惯了用404作为服务器未找到文件的错误代码了。当然实际考证传说中的room 404是不存在的，在http请求3位的返回码中，4开头的代表客户错误，5开头代表服务器端错误。下面是造成404错误的主要原因。

1.您请求的文件已删除。
2.您请求的文件可能已经永久迁移到其他的文件夹。
3.服务器出现错误，无法查找到您所请求的文件。
4.您随便输入了一个地址来调戏服务器。

People are used to using 404 as the error code for the server to find the file. Of course, the actual textual research on the legendary room 404 does not exist. In the 3-bit return code of HTTP request, the beginning of 4 represents the customer error, and the beginning 5 represents the server-side error. The main reason for the error is 404.

The file you requested has been deleted.
2. The file you requested may have been permanently migrated to another folder.
3. There is an error in the server and the file you requested cannot be found.
4. You enter a random address to tease the server.
